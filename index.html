My Projects

QuizApp

Utilised skills include: HTML, JSP, Java, SQL, Agile Development, Model-View-Controller, Git Version Control and Test Driven Development.

Delivrhyno

Utilised skills include: HTML, PHP, SQL, Requirements Analysis, Database Design and Git Version Control.

Minesweeper

Utilised skills include: C# and the Microsoft Visual Studio IDE.

QuizApp is a Quiz Website that allows staff to create quizzes which students can then complete. The website was designed by an agile development team consisting of myself and other students. It makes use of JSP to provide dynamic content with a SQL database storing data including profile information and quiz details.

Delivrhyno is a website developed with a team for a company that offers take-away delivery services to restaurants, whereby customers can place their orders online and have the food delivered directly to them. Users can select from restaurants in their area and choose the food they wish to order. The website also includes a Database Management System, which allows the admin to make manipulations on the database. These manipulations include full CRUD functionality on restaurant employees, branches of the company, restaurants and restaurant menu items. It makes use of PHP for dynamic content with a SQL database.

Minesweeper, as its name would imply, is a clone of the classic computer game minesweeper. It is programmed in C# and uses Windows Forms to provide the GUI. Features include Saved High Scores and GUI elements that make the user aware of how many mines are near the tile that they have just uncovered. To complete the game the user must mark out all mines on the game screen successfully without detonating any mines.
