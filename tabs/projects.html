Delivrhyno

HTML CSS Bootstrap PHP MySQL

Delivrhyno is a website developed with a team for a company that offers take-away delivery services to restaurants, whereby customers can place their orders online and have the food delivered directly to them. Users can select from restaurants in their area and choose the food they wish to order. The website also includes a Database Management System, which allows the admin to make manipulations on the database. These manipulations include full CRUD functionality on restaurant employees, branches of the company, restaurants and restaurant menu items. The website makes use of PHP for dynamic content in conjunction with a SQL database.

Minesweeper

C# WinForms

Minesweeper, as its name would imply, is a clone of the classic computer game minesweeper. It is programmed in C# and uses Windows Forms to provide the GUI. Features include Saved High Scores and GUI elements that make the user aware of how many mines are near the tile that they have just uncovered. To complete the game the user must mark out all mines on the game screen successfully without detonating any mines.
